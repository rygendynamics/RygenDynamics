## 🎓 **Learning & Customization**

### **To Modify 3D Elements:**
1. Edit `src/components/3D/RobotArm3D.jsx` for robot customization
2. Edit `src/components/3D/ParticleField.jsx` for particle effects
3. Use React Three Fiber docs: https://docs.pmnd.rs/react-three-fiber

### **To Change Colors:**
1. Edit `src/index.css` - CSS variables
2. Modify component-specific color values

### **To Add New Sections:**
1. Create new component in `src/components/`
2. Import and add to `src/App.jsx`
